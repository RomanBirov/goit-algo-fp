Висновки щодо правильності розрахунків шляхом порівняння результатів, отриманих за допомогою методу Монте-Карло, з результатами аналітичних розрахунків. Порівняння показало, що зі збільшенням кількості симуляцій імовірності, обчислені методом Монте-Карло, наближаються до теоретичних значень. Це підтверджує коректність реалізації алгоритму та ефективність методу Монте-Карло для оцінювання ймовірностей.
